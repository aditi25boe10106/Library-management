PROJECT TITLE - Library Management System

OVERVIEW OF THE PROJECT - A Library Management System is a software application designed to manage the daily operations of a library. The primary goal of this project is to automatize and computerize the manual activities of a library to improve efficiency, accuracy, and accessibility for both staffs and members.

FEATURES

(1) Administrative Features - Cataloging & Inventory Management - Book/Resource Entry, Barcode/RFID Integration, Stock Verification - Circulation Control - Issue/Check-Out, Return/Check-In, Holds/Reservations, Renewal - Member Management - Registration/Enrollment, Role-Based Access, Suspension/Blocking - Financial & Reporting - Fine Calculation, Fee Recording, Report Generation
(2) Members Features - Online Public Access - Advanced Search, Real-Time Status, Location Finder Self-Service Account Management - Personal Login, Borrowing History, Renewal Request, Place Holds, Alerts & Notifications
TECHNOLOGIES/TOOLS USED

(1) Programming Languages - Python and HTML
(2) Database Technologies - MySQL and SQL(Structured Query Language)
(3) Specialized Tools and Libraries - Git & GitHub/GitLab
(4) Deployment Environment - Web Server and Operating System
STEPS TO INSTALL AND RUN THE PROJECT

Phase 1: Preparation - (1) Install Prerequisites - Install Python, Install MySQL, Install Git (2) Get the Project Code - Clone the Repository, Navigate to Project Folder (3) Create a Virtual Environment - Create Environment, Activate Environment (4) Install Dependencies
Phase 2: Database Setup - (1) Create the Database - Open MySQL Client, Create Database, Exit MySQL (2) Configure the Project - Locate Settings, Update Database Credentials (3) Run Migrations
Phase 3: Run the System - (1) Create Superuser (2) Launch the Server (3) Access the Application
INSTRUCTION FOR TESTING -

(1) Functional Testing - Cataloging, Circulation, Patron Management, Fines, real-time availability status
(2) Integration Testing - Does it Connect
(3) Non-Functional Testing - Security, Performance, Usability
